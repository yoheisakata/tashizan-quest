# アプリひろば 🎮

スマホブラウザで遊べる3つのアプリです。

---

## ⚔️ たしざんクエスト

`tashizan/`

子ども向けの足し算学習ゲーム。計算問題を解いてモンスターを倒し、勇者を育てよう。

- レベルアップ・セーブ機能あり
- 難易度が段階的に上がる
- スマホ完全対応

---

## ♻️ ゴミクエスト

`gomi/`

落ちてくるゴミをタップして集めるアクションゲーム。60秒でどれだけ稼げるか競おう。

- タイムアタック形式（60秒）
- ゴミの種類によって報酬が異なる
- スマホ完全対応

---

## 🇪🇸 Español

`espanol/`

初心者向けスペイン語学習アプリ。

- **AI会話練習** — AIとスペイン語でチャット。全返答に日本語訳付き
- **聞き取り練習** — スペイン語音声を聞いて4択問題に答える

### セットアップ

[Google AI Studio](https://aistudio.google.com/apikey) で API キーを取得（無料）し、アプリ起動時の入力画面に貼り付けてください。キーはブラウザの localStorage にのみ保存されます。
